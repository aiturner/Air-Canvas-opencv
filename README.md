# 🖌️ Real-Time Air Canvas

Draw in the air using colored markers and your webcam!

## 🎯 Features
- Real-time color detection using HSV filtering
- Smooth drawing with motion tracking
- Color switching (Blue, Green, Red)
- Clear canvas with a gesture or key press
- Save your artwork
